    Import and Export Schema

    For both import and export process, the folder must exists both defined as Oracle directories and in the filesystem.

    Import

    Note

    The schema will be created if it doesn't exists, and it should have the same password as the origin schema.

    SQL> impdp sys/pass remap_schema=SOURCE_SCHEMA:DEST_SCHEMA directory=exp_dir dumpfile=DUMP_FILENAME.dmp logfile=impdp_DUMP_FILENAME.log

    Export

    SQL> expdp SCHEMA_TO_EXPORT/PASSWORD directory=exp_dir dumpfile=DUMP_FILENAME.dmp logfile=export_DUMP_FILENAME.log filesize=1g

    Show active databases

    SQL> SELECT INSTANCE_NAME, DATABASE_STATUS, INSTANCE_ROLE from v$instance;
