    S3

    Mount S3 Bucket on linux

    # Install prerequisites
yum install automake fuse fuse-devel gcc-c++ git libcurl-devel libxml2-devel make openssl-devel -y

# Create mountpoint
cd /srv
mkdir s3fs
cd s3fs/

# Obtain s3fs-fuse.
sudo amazon-linux-extras install epel
sudo yum install s3fs-fuse

    Cloudformation

    Curl for cancel jobs

    In a Lambda python code you can print this curl to cancel the CloudFormation Stack, in case it remains in a IN PROGRESS state

    # CF Parameters to build the response in case of Stack blocking
ServiceToken = event['ServiceToken']
StackId = event['StackId']
RequestId = event['RequestId']
LogicalResourceId = event['LogicalResourceId']
ResponseURL = event['ResponseURL']
# CloudFormation CURL response. Uncomment while testing
logger.info('CloudFormation CURL response. Uncomment while testing')
logger.info("curl -H 'Content-Type: ''' -X PUT -d '{ \"Status\": \"SUCCESS\", \"PhysicalResourceId\": \"" + physical_resource_id + "\", \"StackId\": \"" + StackId + "\", \"RequestId\": \"" + RequestId + "\", \"LogicalResourceId\": \"" + LogicalResourceId + "\"}' '" + ResponseURL + "'")
